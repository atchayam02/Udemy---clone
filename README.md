# HTML & CSS Udemy Project

This project is a simple website created as part of a Udemy course using **HTML** and **basic CSS**. It includes a single main page (`index.html`) styled with an external stylesheet (`style.css`). The project demonstrates fundamental web development concepts such as page structure, layout, and basic styling.

## Files
- `index.html` – Main webpage
- `style.css` – Stylesheet for design and layout
